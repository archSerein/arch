first:
	读取 fifo 队列中 deqP 指向的寄存器
	empty 寄存器
notFull:
	full 寄存器
notEmpty:
	empty 寄存器
clear:
	empty 寄存器
	full 寄存器
	enqP, deqP 寄存器
deq:
	fifo 队列
	deqP, empty 寄存器
enq:
	fifo 队列
	enqP, full 寄存器	

f < g	f 发生在 g 之前(g 不能在一个动作中影响 f)
f > g:	g 发生在 f 之前
C:	f 和 g冲突，不能一起调用
CF:	f 和 g 无冲突，互不影响
conflict:
		notEmpty	notFull 	deq	enq	first	clear
	notEmpty  CF		  CF		<	CF	  <	 >
	notFull	  CF		  CF		CF	<	  CF	 >
	deq	  >		  CF		CF	C	  C	 C
	enq	  CF		  >		C	CF	  CF	 C
	first	  >		  CF		C	CF	  CF	 C
	clear	  <		  >		C	C	  C	 CF


notEmpty	<	dep
notEmpty	<	enp
notEmpty	<	first
notFull		<	enq
notFull		<	deq
deq		C	enq

使用编译器属性强制规则触发(fireEveryCycle)
deq 和 enq 只是发起请求，实际的操作在 fireEveryCycle 中
在 fireEveryCycle 处理时，使用 EHR 给 enq, deq, clear分配不同的端口，强制实施此规则的约束
